Taltio Extension Test Page

How to Install & Test the Extension:

  1. Open Firefox and navigate to about:debugging
  2. Click "This Firefox" in the left sidebar
  3. Click "Load Temporary Add-on" button
  4. Navigate to this project folder and select manifest.json
  5. The extension should now be loaded and active

Testing Instructions:

Method 1 - Text Trigger: Type AI: in any input field below to trigger the prompt dropdown

Method 2 - Hotkey: Focus on any input field and press Ctrl+Shift+P to show the prompt dropdown

Method 3 - Context Menu: Highlight any text on this page, right-click, and select "Save as AI Prompt" to save it

First time? You'll need to add prompts via the extension's options page first!

Test Input Fields

Extension Features

  • Text Trigger: Type "AI:" in any input to show prompts
  • Hotkey: Press Ctrl+Shift+P to show prompts
  • Arrow Navigation: Use ↑/↓ to navigate prompt list
  • Enter to Select: Press Enter to insert selected prompt
  • Escape to Close: Press Esc to hide the dropdown
  • Mouse Support: Click prompts to select them

Add Sample Prompts

Before testing, add some sample prompts through the extension's options page:

  • Code Review: "Please review this code for potential improvements, bugs, and best practices."
  • Explain Code: "Please explain what this code does in simple terms."
  • Bug Fix: "Help me identify and fix the bug in this code."

Customization Options

Access the extension's options page to:

  • Change the hotkey combination (default: Ctrl+Shift+P)
  • Modify the text trigger (default: AI:)
  • Add, edit, and delete saved prompts
  • Manage your prompt library
